## 🌟 核心功能
### 物理耦合（V₀ → t）
跃迁积分`t`由晶格深度`V₀`基于深晶格近似公式动态计算，有效`U/t`实时更新，保证物理自洽性。
### 关键可观测量计算
实时展示超流序参量`|⟨a⟩|`、数涨落`Δn`，并自动判断超流/莫特绝缘相态。
### 可证伪实验模式
- 实验模式：空白相图支持点击记录实验点，自主探索相变边界
- 理论模式：显示预计算临界边界（U_c/t ≈ 25），支持实验数据对比验证
### 轻量可交互可视化
- 8×8实空间晶格可视化，鼠标悬停可观测局域粒子密度
- 动量分布（TOF）仿真，直观区分超流/绝缘相的动量特征
- 全参数滑块实时调节，所有物理结果毫秒级刷新

## 📚 物理背景
基于冷原子光晶格量子系统，核心物理模型采用凝聚态物理经典近似方法，关键公式：
1. 跃迁积分：`t ≈ (4/√π) * V₀^(3/4) * exp(-2√V₀)`（深晶格近似，能量单位Er=1）
2. 超流序参量：`|⟨a⟩| = ⟨√n⟩`（平均场近似，反映超流特性）
3. 数涨落：`Δn = √(⟨n²⟩ - ⟨n⟩²)`（区分超流/绝缘相的关键指标）
4. 动量分布：结构因子`S(k) = |∑ e^(ikr) n_r|²`（TOF飞行时间仿真）
5. 相变临界边界：`U_c/t ≈ 25`（单位填充条件下，超流→莫特绝缘相变点）

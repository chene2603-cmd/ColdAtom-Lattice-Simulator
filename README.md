# ColdAtom Simulator

> 高安全性冷原子物理仿真训练服务平台（v3.0）  
> 支持远程 API 调用、模型自动保存与 Web 可视化，专为科研与教学场景设计

本项目提供一套轻量级、高安全性的冷原子系统仿真服务，支持通过标准 HTTP 接口提交强化学习或物理仿真训练任务。系统内置多重防护机制，确保计算资源不被滥用，数据路径不被越权访问，适用于实验室部署、课程教学及科研协作。

---

## 🔒 安全架构

为保障服务稳定与数据隔离，系统实施以下安全策略：

| 安全措施 | 技术实现 | 防护目标 |
|--------|--------|--------|
| **API Token 认证** | 所有请求需携带 `Authorization: Bearer <token>` | 仅授权用户可提交任务 |
| **凭证环境隔离** | Token 通过 `.env` 加载，**禁止硬编码** | 避免敏感信息泄露至代码仓库 |
| **训练步数限流** | 限制 `100 ≤ timesteps ≤ 1,000,000` | 防止恶意超长任务耗尽 GPU/CPU 资源 |
| **文件名白名单校验** | 仅允许 `[a-zA-Z0-9_-]+\.zip` 格式 | 阻断路径遍历攻击（如 `../../../etc/passwd`） |
| **路径沙箱锁定** | 使用 `Path.resolve()` + 目录相对性校验 | 防御符号链接（symlink）与目录逃逸 |
| **健康状态监控** | 提供 `/health` 接口 | 支持自动化运维与服务探活 |

> 所有安全逻辑均开源可审计，符合科研软件安全开发规范。

---

## 🚀 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置安全凭证
cp .env.example .env
# 编辑 .env，填入强随机 Token（建议：openssl rand -hex 32）

# 3. 初始化目录
mkdir -p models logs static

# 4. 启动服务
uvicorn app.main:app --host 0.0.0.0 --port 8000
